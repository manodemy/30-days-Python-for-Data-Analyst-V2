1. Text Manipulation : Concatenation, Extraction and Casing

🔍 Concept
CONCAT(A1, " ", B1) or A1&" "&B1 joins text values. LEFT(text, n) extracts the first n characters. RIGHT(text, n) extracts the last n. MID(text, start, length) extracts from a middle position. LEN(text) returns the total character count. FIND(substring, text) returns the position of a character.

📊 Day 07 Executive Summary

Text functions are key for data transformation. The ampersand & operator and CONCAT() join strings. LEFT()/RIGHT()/MID() extract parts. A powerful combo: =LEFT(A1, FIND(" ", A1)-1) extracts the first name from any full name.
